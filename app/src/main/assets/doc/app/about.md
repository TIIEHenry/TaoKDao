

### What is TaoKDao?


* An editing platform similar to VSCode, Idea and Notepad++ on a computer


### What are its characteristics?
* Editor platform
* Resource manager
* Powerful plug-in support and management
* High-performance editor
* Multi-language support
* Code formatting
* Auto-completion
* Efficient editing
* Building project
* Custom project
* Custom packaging script
* Multi-window
* Night mode
* comes with Log viewer * etc.


### What it can do?
* Use plugins
  * Plugins provide many functions
  * Plugins are developed with open API
* Edit code files
  * High performance editor
  * Multiple programming languages support
  * Code formatting
  * Code automatic completion
  * Efficient editing
* Build your project
  * Define your own project
  * Define your own action
  * Write your own build scripts
* Best plug-in extension capabilities
  * Support plug-in development
  * Support package plug-in to APK
* Excellent interface and operation experience
  * Support dark mode
  * Support keymap
  * Enjoy coding
